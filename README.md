#  :mortar_board: [졸업프로젝트] 
# React와 Node.js를 이용한 글-고운 웹사이트

###  :bouquet: 프로젝트 내용과 개요

글-고운 사이트란 무엇인가?
```
고민이 있어 위로 받고 싶은 사람들을 위한 웹사이트로
자신의 고민에 대한 선택 버튼을 고르고 나면
최종적으로 자신에게 맞는 위로의 글이 나오는 사이트이다.
```
